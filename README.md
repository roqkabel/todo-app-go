## Todo API in GO

This public repo demonstrate a todo app API development using Go lang.

### Features include.

Todo:

- [x] Create Todo 
- [x] List All Todos 
- [x] Edit Todo 
- [x] Delete Todo 
- [x] Toggle Todo Is Completed / Not Completed
- [ ] Attach User ID to Todos.
- [ ] Prevent Unauthorized users from creating Todo
  

Users:

- [x] Create New User 
- [x] Login User
- [x] JWT Token Generation
